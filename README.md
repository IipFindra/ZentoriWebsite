\# Sistem Inventori Barang (PHP Native)



Sistem Inventori Barang ini dibuat menggunakan PHP Native tanpa framework. 

Fitur utama:

\- Manajemen barang (tambah, edit, hapus)

\- Pencatatan stok masuk \& keluar

\- Manajemen supplier

\- Dashboard ringkas

\- Login \& autentikasi pengguna

\- Notifikasi stok minimum
